## 🧠 Project Overview  
This is a full-stack task management application that allows users to create, update, delete, and filter tasks. The app includes user authentication and session management, with all data stored in a PostgreSQL database.

---

## 🚀 Features  
- User authentication (login & session management)  
- Create, update, and delete tasks (CRUD)  
- Task filtering functionality  
- Persistent data storage with PostgreSQL  
- Demo account with automatic task reset for clean testing  

---

## 🛠️ Tech Stack  
**Frontend:** HTML, CSS, JavaScript  
**Backend:** Node.js, Express  
**Database:** PostgreSQL  
**Deployment:** Render
